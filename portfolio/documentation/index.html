Getting Started

Unzip files from archive and you will find Ariox Html folder. Inside you'll find following files and folders:

  1. /css - folder with css files
  2. /fonts - folder with icon fonts
  3. /img - folder with all images
  4. /js - folder with jQuery script files html files send-mail.php - file for contact form
Note: Contact form will not work offline. You must upload your website on a server with support of PHP and PHP mail() function.

Basic Html Structure

Basic Html Sctructure of any section looks like this:

<section class="section id">
    <div class="container"> 
        <h2 class="title">Section Heading</h2>
 
        <div class="row">
            <div class="col-md-3">
            ...
            </div> <!-- end col -->
        </div> <!-- end row -->
 
    </div> <!-- end container -->
</section> <!-- end section -->

Bootstrap Grid

Template is based on Bootstrap Framework. Using a single set of .col-md-* grid classes, you can create a basic grid system that starts out stacked on mobile devices and tablet devices (the extra small to small range) before becoming horizontal on desktop (medium) devices. Place grid columns in any .row. If you want to learn more about Bootstrap Grid System, please visit this link

2 equal columns:

<div class="row">
    <div class="col-md-6">.col-md-6</div>
    <div class="col-md-6">.col-md-6</div>
</div>

3 equal columns:

<div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4">.col-md-4</div>
</div>

8 left columns + 4 right columns:

<div class="row">
    <div class="col-md-8">.col-md-8</div>
    <div class="col-md-4">.col-md-4</div>
</div>

Portfolio

Portfolio filter has the following structure:

<ul class="portfolio-filter">
   <li><a href="#" class="filter active" data-filter="*">All</a></li>
   <li><a href="#" class="filter" data-filter=".branding">Branding</a></li>
   <li><a href="#" class="filter" data-filter=".illustration">Illustration</a></li>
   <li><a href="#" class="filter" data-filter=".web-design">Web Design</a></li>
   <li><a href="#" class="filter" data-filter=".print">Print</a></li>
</ul>

Data-filter is the name of the categories. Categories of the works should be defined here

Here is how portfolio item structure looks: 
<div class="masonry-item branding">
   <div class="thumb">
      <div class="overlay"></div>
      <img src="img/portfolio/01.jpg" alt="" />
      <div class="meta">
         <div class="inner">
            <div class="btns">
               <a href="img/portfolio/large/01.jpg" data-effect="mfp-zoom-in"><i class="ion-search"></i></a>
               <a href="img/portfolio/large/02.jpg" data-effect="mfp-zoom-in"><i class="ion-link"></i></a>
            </div>
            <h4>Portfolio Title</h4>
         </div>
      </div>
   </div>
</div>
